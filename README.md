# Règles du jeu
### Les bases
Le jeu en question est un jeu de carte multijoueurs qui se déroule tour par tour. Chaque joueur a un deck identique comprenant 5 personnages différents en 5 exemplaires chacun. Chaque joueur possède :
-   une main (_hand_) : pouvant contenir jusqu’à 5 cartes maximum. Elle ne peut être vue que par son propriétaire.
-   une pile de bibliothèque (_librairy_) : que personne ne peut voir.
-   une pile de défausse (_graveyard_) : qui peut être vue par tous les joueurs. Les cartes n’y ont aucun effet.
-	un champs de bataille (_battlefield_) : qui peut être vu par tous les joueurs.

### Mise en place
Mélangez votre deck et piochez 5 cartes. Le joueur qui joue le premier ne pioche pas de carte à l’étape de pioche.

### Déroulement d’un tour
1.  **Phase de pioche –** Piochez une carte dans votre bibliothèque.    
2.  **Phase principale –**  Vous pouvez jouer une carte de votre main en la mettant sur le champs de bataille. Chaque personnage a une capacité spécifique qui peut être activée ou non à son arrivée sur le champs de bataille (voir tableau ci-dessous).
3.  **Phase de fin –** déclarez la fin de votre tour.
    
### Fin du jeu
Le jeu se termine lorsqu’un joueur a gagné. Vous gagnez si vous parvenez à réunir, sous votre contrôle, les cinq personnages sur le champs de bataille. Vous perdez si un autre joueur gagne ou si vous devez piochez alors que le nombre de carte dans votre bibliothèque est nul.

### Invoquer des personnages
| **Types de personnages** | **Capacités** |
|--|--|
| Héraut (_Herald_) | Piochez une carte. |
| Assassin (_Murderer_) | Assassinez le personnage ciblé. |
| Sorcière (_Witch_) | Ciblez un joueur, il se défausse d’une carte. |
| Nécromancien (_Necromancer_) | Ressussitez un personnage depuis votre cimetière. |
| Magicien (_Wizard_) | Défaussez vous d’un carte et contrecarrez le personnage ciblé. (Pour activer cette capacité le Magicien doit être joué pendant la phase principale d’un adversaire) |

### Dictionnaire :
-   **Piocher [X carte(s)]** : prendre les X premières carte de la bibliothèque pour la/les mettre dans la main.
-   **Assassiner [un personnage]** : depuis le champs de bataille, placer le personnage ciblé dans le cimetière de son propriétaire.
-   **Ressussiter [un personnage]** : renvoyer le personnage ciblé du cimetière de son propriétaire dans sa main.
-   **(Se) défausser [de X carte]** : le joueur ciblé doit retirer X carte(s) de sa main pour la/les placer dans son cimetière.
-   **Contrecarrer [un  personnage]** : cibler un personnage qui vient d’être mis en jeu et le placer dans le cimetière de son propriétaire. La capacité du personnage ciblé n’est donc pas activée.
